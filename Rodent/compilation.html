Запуск программы в Rodent

Для запуска программы нужно собрать проект

Чтобы собрать проект, выполните следующие команды в терминале:

mkdir build; cd build
cmake ..
make

После этого в директории build будет создан исполняемый файл Rodent.

Синтаксис запуска программы

Для запуска программы используйте следующий синтаксис:

./Rodent ../path/to/file.rod
./Rodent -l aeaya.dev ../path/to/file.rod

Где ../path/to/file.rod — путь к файлу с исходным кодом на языке Rodent.
