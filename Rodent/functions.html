Функции

Объявление функций

Общий вид объявления функции:

func a() -> type {
    body();
    
    return value; // if not void
}

Примеры:

func array_sum(array a, int size) -> int {
  int result = 0;
  for (int i = 0; i < size; ++i){
    result += a[i];
  }
  return result;
}

func f() -> double {
    return 0.1 + 0.2;
}

func f(double e) -> void {
    int a = 1;
    double c = a + e;
}

Вызов функции

Общий вид вызова функции:

type a = function_with_return_type_same_as_variable_a(1, 2, 3);
type b = function_with_return_type_same_as_variable_b();

void_function(0);

Примеры:

array a = [1, 6, 23, -2];
int arr_sum = array_sum(a, 4);

str string = "Rodent is the best language";
reverse_string(string);
