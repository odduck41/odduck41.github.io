Создание переменных

Общий вид

Переменные создаются с помощью оператора типа. В общем случае это выглядит так:

type name = value;
type name1, name2, name3;
type name1 = value1, name2, name3 = value3;

Примеры

Вот несколько примеров создания переменных:

int a = 5;
int b;
char c = 'r';
bool is_rodent_is_the_best_lang = true;
double d = 0.213;

Для строк:

str s = "Rodent is the best programming language";

Для массивов:

array arr = [2, 2, 6, -25];
