Циклы

Цикл for

Цикл for используется для повторения кода фиксированное количество раз. Он состоит из трёх частей: начальной настройки, условия завершения и шага, выполняемого после каждой итерации.

Общий вид цикла for:

for (<first_action>; <bool_exit_condition>; <action_after_iteration>) {
    <body>
}

Пример:

array<int> a = [1, 2, 3, 4];

for (int i =  0; i < 4; ++i) {
    ++a[i];

    if (a[i] == 4){
        break;
    }
}

Цикл while

Цикл while выполняет блок кода до тех пор, пока заданное условие истинно.

Общий вид цикла while:

while (<condition>) {
    <body>
}

Пример:

int a = 1;
while (a < 10) {
    if (a == 3) {
        break;
    }
    a *= 2;
}

Цикл do-while

Цикл do-while гарантирует выполнение блока кода хотя бы один раз.

Общий вид цикла do-while:

do {
    <body>
} while (<condition>)

Пример:

int a = 1;
do {
    a += 1;
} while (a != 2);
